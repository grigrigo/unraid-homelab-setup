# 📚 사전 요구사항

Unraid NAS를 성공적으로 구축하기 위해 필요한 기술과 지식

---

## 🎯 필수 기술

### 1. 컴퓨터 하드웨어 기초
- ✅ PC 부품 식별 능력 (CPU, RAM, HDD, SSD)
- ✅ SATA 케이블 연결
- ✅ 전원 케이블 연결
- ✅ BIOS/UEFI 접근 방법

**학습 자료**:
- PC 조립 가이드 (YouTube)
- 컴퓨터 부품 기초 강의

### 2. 기본 네트워킹
- ✅ IP 주소 이해 (192.168.x.x)
- ✅ 라우터 관리 페이지 접근
- ✅ 이더넷 케이블 연결
- ✅ 기본 포트 개념

**필요 지식**:
```
- IP 주소: 컴퓨터의 네트워크 주소
- 서브넷: 255.255.255.0
- 게이트웨이: 라우터 IP (보통 192.168.1.1)
- DNS: 8.8.8.8 (Google DNS)
```

### 3. 기본 Linux 명령어
최소한 다음 명령어는 알아야 함:
```bash
ls          # 파일 목록
cd          # 디렉토리 이동
cp          # 파일 복사
mv          # 파일 이동
rm          # 파일 삭제
mkdir       # 디렉토리 생성
chmod       # 권한 변경
df -h       # 디스크 사용량
```

## 📖 필요한 배경 지식

### 스토리지 개념
- **HDD vs SSD**: 속도와 용량의 차이
- **패리티**: 데이터 보호 방식
- **캐시**: 빠른 쓰기를 위한 중간 저장소
- **SMART**: 디스크 상태 모니터링

### 네트워크 개념
- **SMB/CIFS**: Windows 파일 공유
- **NFS**: Linux 파일 공유
- **포트 포워딩**: 외부 접속 허용
- **VPN**: 안전한 원격 접속

### 보안 개념
- **2FA**: 2단계 인증
- **SSH 키**: 비밀번호 없는 인증
- **SSL/TLS**: 암호화된 연결
- **방화벽**: 네트워크 접근 제어

## 🔍 기술 부족 시 대안

### Linux 명령어 모르는 경우
➡️ Unraid 웹 UI로 대부분 작업 가능
➡️ Community Apps로 쉬운 설치
➡️ 필요시 포럼에서 명령어 복사/붙여넣기

### Docker 모르는 경우
➡️ Community Apps 템플릿 사용
➡️ 미리 구성된 스택 사용
➡️ YouTube 튜토리얼 따라하기

### 네트워크 설정 어려운 경우
➡️ 기본 설정으로 시작
➡️ DHCP 자동 할당 사용
➡️ 고급 기능은 나중에 추가

## ✅ 준비 상태 체크리스트

### 필수 체크
- [ ] BIOS 진입 방법을 안다
- [ ] IP 주소가 무엇인지 안다
- [ ] 파일을 복사/이동할 수 있다
- [ ] 웹 브라우저를 사용할 수 있다

### 권장 체크
- [ ] SSH 접속을 해본 적이 있다
- [ ] Docker를 들어본 적이 있다
- [ ] 포트 번호가 무엇인지 안다
- [ ] 백업의 중요성을 이해한다
