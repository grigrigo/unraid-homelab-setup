## 🎯 빠른 체크리스트

### 필수 확인 사항
- [ ] 기본적인 PC 조립 경험이 있는가?
- [ ] Linux 명령줄을 다룰 수 있는가?
- [ ] 네트워크 기본 지식이 있는가?
- [ ] 최소 70,000원의 초기 예산이 있는가?
- [ ] 24시간 운영 가능한 공간이 있는가?

### 하드웨어 준비
- [ ] Intel 또는 AMD 64비트 CPU
- [ ] 최소 8GB RAM (권장 16GB+)
- [ ] 최소 3개의 HDD
- [ ] 1개의 SSD (캐시용)
- [ ] 4-32GB USB 드라이브

## ⚡ 프로젝트 하이라이트

### 구축 후 가능한 것들
- ✅ **9TB** 보호된 스토리지 (싱글 패리티)
- ✅ **5-8개** 동시 미디어 스트리밍
- ✅ **50+** Docker 컨테이너 운영
- ✅ **자동** 백업 시스템
- ✅ **원격** 접속 가능
- ✅ **월 12,000원** 이하 전기료

### 주요 기능
- 🎬 **미디어 서버** - Plex/Jellyfin
- 📸 **사진 백업** - Google Photos 대체
- ☁️ **개인 클라우드** - Dropbox 대체
- 🔒 **보안** - VPN, 암호화, 2FA
- 📊 **모니터링** - 실시간 상태 확인
- 💾 **백업** - 3-2-1 전략 구현

## ⏱️ 예상 소요 시간

| 단계 | 소요 시간 | 난이도 |
|------|-----------|--------|
| 계획 | 2-4시간 | ⭐ |
| 하드웨어 준비 | 1-2일 | ⭐ |
| 초기 설치 | 4-6시간 | ⭐⭐ |
| 패리티 동기화 | 6-24시간 | ⭐ |
| 앱 설정 | 2-3일 | ⭐⭐ |
| 최적화 | 1주일 | ⭐⭐⭐ |

**총 소요 시간**: 약 2-3주 (여유있게)

## 🚫 시작하기 전 경고

### ⚠️ 주의사항
- 데이터 손실 가능성이 있는 작업들이 포함됨
- 잘못된 설정은 하드웨어 손상 가능
- 백업 없이 진행하지 말 것
- 전기 안전 주의 필요

### 🛑 이런 경우 진행 중단
- USB 부팅이 안 되는 경우
- BIOS에서 AHCI 모드 지원 안 함
- 네트워크 연결 불가
- 디스크 SMART 오류 발견
