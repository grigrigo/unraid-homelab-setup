# ✅ 설치 검증 체크리스트

Unraid 설치 각 단계별 검증 포인트와 품질 확인

---

## 📋 검증 개요

각 설치 단계를 완료한 후 반드시 확인해야 할 검증 포인트입니다. 체크리스트를 통과하지 못하면 다음 단계로 진행하지 마세요.

## 🔍 단계별 검증 체크포인트

## Stage 1: 하드웨어 준비 검증

### ✓ 하드웨어 테스트 완료
```bash
[ ] Memtest86+: 2 pass, 0 errors
[ ] 모든 디스크 SMART: PASSED
[ ] 배드블록 스캔: 0 bad blocks
[ ] CPU 스트레스: 4시간 무오류
[ ] 전원 전압: ±5% 이내
```

### ✓ 측정값 기록
```markdown
테스트 날짜: ___________

메모리:
- 용량: _____ GB
- 속도: _____ MHz
- 오류: 0

디스크 (각 디스크별):
- 모델: ___________
- 용량: _____ TB
- SMART: PASSED
- 온도: _____ °C

시스템:
- CPU 최고 온도: _____ °C
- 12V 전압: _____ V
- 5V 전압: _____ V
```

### 🚫 진행 중단 조건
- 메모리 오류 1개 이상
- SMART 실패 디스크
- 배드블록 발견
- CPU 온도 > 85°C
- 전압 변동 > ±5%

## Stage 2: BIOS 설정 검증

### ✓ 필수 설정 확인
```bash

[ ] Boot Mode: UEFI
[ ] Secure Boot: Disabled
[ ] CSM: Disabled
[ ] USB Boot: Priority #1
```

### ✓ 성능 설정 확인
```bash
[ ] VT-x/AMD-V: Enabled (VM 사용 시)
[ ] VT-d/IOMMU: Enabled (패스스루 시)
[ ] C-States: Enabled (절전)
[ ] Turbo Boost: Enabled
[ ] XMP/DOCP: 설정 완료 (선택)
```
